\chapter{射电信号的探测方法}

射电(10MHz--1THz) $\ne$无线电(3kHz--3GHz).

白天可测: 波长远大大气尘埃, 无散射; 太阳射电信号少.

排除水汽: 高地, 旱地.

\begin{itemize}
    \item 镜面精度低($\lambda/20$).
    \item $\lambda$大, 在$\lambda^3$范围内放很多带点粒子, 形成相干辐射.
    \item 波长远大星际尘埃, 透明.
    \item $h\nu/kT\ll1$, 所有天体都辐射.
    \item $\theta\sim\lambda/D$, 需要大$D$.
\end{itemize}
